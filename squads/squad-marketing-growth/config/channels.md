# Canais de Aquisição — squad-marketing-growth

## Princípio Fundamental

Esteja onde o jardineiro está. Não espere que ele venha até você.

---

## Tier 1 — Canais Primários (60% do budget)

### Referral (Indicação Peer-to-Peer)
| Atributo | Valor |
|----------|-------|
| Budget share | 15% (custo do mês grátis) |
| CAC estimado | R$15-25 |
| Time to result | Imediato (após base ativa) |
| Escala | Alta (viral coefficient) |
| Responsável | garden-referral |

**Mecânica:** "Indique 1 amigo = 1 mês grátis pros dois"
**Por que #1:** Jardineiro confia em jardineiro. Menor CAC. Maior retenção do indicado.
**KPIs:** Viral coefficient (K), share rate, referral conversion rate

---

### WhatsApp Viral
| Atributo | Valor |
|----------|-------|
| Budget share | 10% (produção de cards) |
| CAC estimado | R$5-15 |
| Time to result | Imediato |
| Escala | Alta (grupos de 50-200 membros) |
| Responsável | garden-content |

**Mecânica:** Cards compartilháveis, mensagens prontas, dicas visuais
**Por que:** É o "feed" do jardineiro. Se é bom, ele encaminha.
**KPIs:** Forward rate (proxy: cliques), downloads via link WhatsApp

---

### Instagram Reels
| Atributo | Valor |
|----------|-------|
| Budget share | 20% (produção + boost ocasional) |
| CAC estimado | R$20-40 |
| Time to result | 2-4 semanas |
| Escala | Média-Alta |
| Responsável | garden-content |

**Mecânica:** Reels de 30s: antes/depois de jardins, dicas de negócio, testimonials
**Por que:** Jardineiro já usa Instagram para mostrar trabalho. Formato visual é natural.
**Frequência:** 3 Reels/semana + 1-2 posts estáticos
**Horário:** 18h-20h (jardineiro após expediente)
**KPIs:** Saves, shares, profile visits, link clicks

---

### Parcerias com Lojas de Insumos
| Atributo | Valor |
|----------|-------|
| Budget share | 15% (material + comissão) |
| CAC estimado | R$20-35 |
| Time to result | 30-60 dias (setup) |
| Escala | Média (limitada por lojas na região) |
| Responsável | garden-referral |

**Mecânica:** Cartaz no caixa, cartão de visita, recomendação do dono
**Por que:** Jardineiro vai à loja toda semana. Dono da loja é o influencer #1.
**KPIs:** Downloads por loja/mês, parceiros ativos

## Tier 2 — Canais Secundários (30% do budget)

### YouTube Shorts
| Atributo | Valor |
|----------|-------|
| Budget share | 10% |
| CAC estimado | R$25-50 |
| Time to result | 4-8 semanas |
| Escala | Média |
| Responsável | garden-content |

**Mecânica:** Shorts de 60s com dicas de negócio para jardineiros
**Frequência:** 2/semana
**KPIs:** Watch time, CTR para link na bio

---

### Google Local SEO
| Atributo | Valor |
|----------|-------|
| Budget share | 10% (produção de conteúdo) |
| CAC estimado | R$10-20 (orgânico, longo prazo) |
| Time to result | 3-6 meses |
| Escala | Alta (acumula) |
| Responsável | garden-content |

**Mecânica:** Blog posts otimizados para "app para jardineiros", "gestão jardinagem"
**Frequência:** 2 artigos/mês
**KPIs:** Posição no Google, organic traffic, downloads via blog

---

### Micro-Influencers (Jardineiros Reais)
| Atributo | Valor |
|----------|-------|
| Budget share | 10% |
| CAC estimado | R$30-60 |
| Time to result | Imediato (por campanha) |
| Escala | Baixa-Média |
| Responsável | garden-marketing-chief |

**Mecânica:** Jardineiros com 1K-10K seguidores mostrando uso real do app
**Por que:** Autenticidade > alcance. Jardineiro real > influencer lifestyle.
**KPIs:** Downloads atribuídos, engajamento no post

## Tier 3 — Canais Experimentais (10% do budget)

### Associações e Cooperativas
- Descontos coletivos, workshops presenciais
- Budget: 5%
- Timeline: 60-90 dias para ativar

### Feiras e Eventos de Jardinagem/Paisagismo
- Estande simples, demonstração ao vivo
- Budget: 5%
- Timeline: sazonal (2-3 eventos/ano)

---

## Regras de Realocação

- Se canal performar 50% abaixo do target por 2 semanas → realocar 50% do budget para melhor canal
- Se canal performar 2x acima do target → aumentar budget em 30% do canal experimental
- Revisão de mix de canais: mensal
- Revisão completa de estratégia: trimestral
