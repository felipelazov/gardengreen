# Assessments

This directory stores axioma assessment outputs generated by the `pv-axioma-assessment` task.

## File Naming

Files are named using the pattern: `{squad-name}-{date}.md`

Example: `squad-creator-pro-2026-03-15.md`

## Tracking

Assessment outputs are ephemeral and not tracked in git. The `.gitkeep` file exists only to preserve the directory structure.
