# Heuristic: Data Before Opinion

- **ID:** TF_MA_001
- **Category:** Market Analysis
- **Source:** [SOURCE: Thiago Finch, Método de Análise]

## Rule
QUANDO fazendo recomendação estratégica, SEMPRE apresentar dados de mercado primeiro, opinião depois.

## Rationale
Opinião sem dados é achismo. Dados sem opinião é planilha. A combinação — dados + interpretação estratégica — é o que gera valor.

## Application
1. Coletar dados quantitativos (tamanho mercado, competidores, trends)
2. Coletar dados qualitativos (dores, feedback, reviews)
3. Interpretar: o que os dados dizem?
4. Recomendar: baseado nos dados, a recomendação é X

## Quick Data Sources (5 min)
- Google Trends → validar interesse temporal
- LinkedIn → contar profissionais no nicho
- Product Hunt → competidores digitais
- Reddit/forums → dor real do mercado
- Glassdoor/reviews → satisfação com soluções atuais

## Anti-Pattern
"Eu acho que o mercado está crescendo" → ❌
"Google Trends mostra 340% de crescimento em 24 meses para [termo]" → ✅

## Veto Condition
Recomendação sem pelo menos 1 dado quantitativo → VETO.
