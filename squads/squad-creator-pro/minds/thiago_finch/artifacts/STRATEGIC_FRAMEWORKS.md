# Strategic Frameworks: Thiago Finch

## Primary Framework: Strategic Positioning Canvas

### Purpose
Posicionar qualquer produto/serviço/squad no mercado de forma defensável e escalável.

### Steps
1. **MAP** — Mapear o domínio: players, tamanho, tendências
2. **GAP** — Identificar gaps: o que ninguém faz bem
3. **VALUE** — Definir proposta de valor: qual transformação entrega
4. **POSITION** — Posicionar: espaço mental claro e defensável
5. **VALIDATE** — Validar: dados confirmam a oportunidade?
6. **MONETIZE** — Monetizar: modelo de receita adequado

### When to Use
- Antes de criar qualquer squad novo
- Ao avaliar viabilidade de domínio
- Ao definir posicionamento de squad comercial

## Secondary Framework: Competitive Moat Analysis

### Purpose
Avaliar sustentabilidade do posicionamento ao longo do tempo.

### Dimensions
| Dimension | Question | Score (1-5) |
|-----------|----------|-------------|
| Switching Cost | Quão difícil é trocar para competidor? | ___ |
| Network Effects | Mais usuários = mais valor? | ___ |
| Brand Moat | A marca é reconhecida no nicho? | ___ |
| Technology Moat | Tecnologia é difícil de replicar? | ___ |
| Data Moat | Dados proprietários geram vantagem? | ___ |

### Interpretation
- Score >= 20: Moat forte — posição defensável
- Score 15-19: Moat moderado — investir em fortalecimento
- Score < 15: Moat fraco — risco de commoditização

## Tertiary Framework: Value Stack

### Purpose
Construir oferta irresistível empilhando valores.

### Steps
1. Listar todas as transformações que o produto entrega
2. Atribuir valor percebido a cada transformação
3. Empilhar do menor ao maior valor
4. Somar valores: Total de Valor Percebido
5. Definir preço como 10-20% do valor total

### Example
| Transformação | Valor Percebido |
|---------------|----------------|
| Templates prontos | R$500 |
| Metodologia estruturada | R$2,000 |
| Expert-level output | R$5,000 |
| Economia de tempo (40h/mês) | R$8,000 |
| **Total** | **R$15,500** |
| **Preço sugerido** | **R$1,500-3,000** |

## Decision Framework: ROI Gate

### Purpose
Toda decisão de investimento passa pelo ROI Gate.

### Formula
```
ROI = (Valor Gerado - Custo Investido) / Custo Investido × 100
```

### Decision Rules
| ROI | Decision |
|-----|----------|
| > 300% | GO — investir imediatamente |
| 100-300% | GO — investir com monitoramento |
| 50-100% | REVIEW — validar premissas |
| 0-50% | CAUTION — buscar alternativas |
| < 0% | NO-GO — não investir |

### Time-to-Value Modifier
- ROI alto + time-to-value curto = PRIORIDADE MÁXIMA
- ROI alto + time-to-value longo = INVESTIR COM PACIÊNCIA
- ROI baixo + time-to-value curto = QUICK WIN
- ROI baixo + time-to-value longo = ELIMINAR
