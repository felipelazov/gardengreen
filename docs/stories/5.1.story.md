## Story

**As a** jardineiro que concluiu um servico,
**I want** gerar uma cobranca PIX para o cliente,
**so that** possa receber o pagamento de forma rapida e rastreavel.

## Acceptance Criteria

```gherkin
AC-1: Gerar PIX com sucesso
GIVEN o servico esta com status "completed"
WHEN o jardineiro toca em "Cobrar PIX"
THEN o sistema chama a Edge Function generate-pix
AND a cobranca e criada no Asaas com billingType=PIX
AND um registro de pagamento e salvo localmente com status=pending
AND o QR Code e exibido em um modal

AC-2: Exibir modal de cobranca PIX
GIVEN o PIX foi gerado com sucesso
WHEN o modal e aberto
THEN o QR Code e exibido como imagem (base64)
AND o campo "Copia e cola" e exibido com botao de copiar
AND o temporizador de expiracao de 24h e exibido
AND o botao "Enviar link por WhatsApp" esta disponivel
AND o botao "Marcar como pago" (fallback) esta disponivel

AC-3: Enviar link por WhatsApp
GIVEN o PIX foi gerado e o modal esta aberto
WHEN o jardineiro toca em "Enviar link por WhatsApp"
THEN o app abre o WhatsApp com mensagem pre-formatada em PT-BR
AND a mensagem contem o link de pagamento e o valor

AC-4: Confirmacao via webhook
GIVEN o cliente realizou o pagamento
WHEN o Asaas envia evento PAYMENT_RECEIVED ao webhook
THEN o pagamento e atualizado para status=paid no banco
AND uma push notification e enviada ao jardineiro: "Pagamento recebido!"

AC-5: Erro ao gerar PIX
GIVEN uma falha na API do Asaas ou na Edge Function
WHEN o usuario toca em "Cobrar PIX"
THEN o sistema exibe: "Nao foi possivel gerar o PIX. Tente novamente."
AND o estado do pagamento nao e alterado
```

[Source: PRD Section EPIC-05, Story 5.1]

## Dev Notes

### Env Vars Necessarias

```bash
ASAAS_API_KEY=your-api-key
ASAAS_ENVIRONMENT=sandbox # ou production
```

### Asaas API Reference

- Base sandbox: `https://sandbox.asaas.com/api/v3`
- Base producao: `https://api.asaas.com/v3`
- Header: `access_token: $ASAAS_API_KEY`
- Criar cobranca: `POST /v3/payments`
- Buscar QR Code: `GET /v3/payments/{id}/pixQrCode`

### Campos do payment record

```typescript
{
  service_id, user_id, client_id,
  amount,          // em reais
  method: 'pix',
  status: 'pending',
  pix_transaction_id: asaasPaymentId,
  pix_link: link
}
```
