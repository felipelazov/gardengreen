## Story

**As a** jardineiro que precisa gerenciar suas financas,
**I want** ver todos os pagamentos pendentes em uma lista clara,
**so that** saiba exatamente quanto tenho a receber e possa agir rapidamente.

## Acceptance Criteria

```gherkin
AC-1: Exibir total pendente
GIVEN existem pagamentos com status=pending
WHEN o jardineiro acessa a aba Financeiro
THEN o total a receber e exibido em destaque no topo: "R$ X.XXX a receber"
AND o valor e formatado em BRL com 2 casas decimais

AC-2: Lista ordenada por antiguidade
GIVEN existem pagamentos pendentes de diferentes datas
WHEN a lista e carregada
THEN os pagamentos sao exibidos do mais antigo ao mais recente (oldest first)
AND cada item exibe: nome do cliente, data do servico, valor, dias pendentes

AC-3: Acoes rapidas por item
GIVEN a lista de pendentes esta exibida
WHEN o jardineiro ve um item
THEN o botao "Cobrar PIX" gera uma nova cobranca PIX (abre PixModal)
AND o botao "Marcar pago" abre o ManualPaymentModal

AC-4: Estado vazio
GIVEN nao existem pagamentos pendentes
WHEN o jardineiro acessa a tela
THEN a mensagem "Nenhuma cobranca pendente!" e exibida
AND um icone comemorativo (ex.: emoji 🎉) e exibido

AC-5: Carregamento e refresh
GIVEN a lista foi carregada
WHEN o jardineiro puxa a lista para baixo (pull-to-refresh)
THEN a lista e recarregada com os dados mais recentes
```

## Dev Notes

### Formatacao de Moeda

```typescript
export function formatBRL(value: number): string {
  return new Intl.NumberFormat('pt-BR', {
    style: 'currency',
    currency: 'BRL',
  }).format(value);
}
```

### Calculo de Dias Pendentes

```typescript
export function daysPending(createdAt: Date): number {
  const today = new Date();
  const diff = today.getTime() - createdAt.getTime();
  return Math.floor(diff / (1000 * 60 * 60 * 24));
}
```
