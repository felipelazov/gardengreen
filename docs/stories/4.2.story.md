## Story

**As a** jardineiro,
**I want** enviar o orcamento formatado via WhatsApp,
**so that** o cliente receba uma proposta profissional diretamente no celular dele.

## Acceptance Criteria

```gherkin
AC-1: Gerar mensagem formatada
GIVEN um orcamento com status "Rascunho" ou "Enviado"
WHEN o jardineiro toca em "Enviar por WhatsApp"
THEN uma mensagem formatada e gerada com emojis, itens listados e total em destaque
AND o numero do orcamento aparece na mensagem

AC-2: Abrir WhatsApp com deep link
GIVEN a mensagem gerada
WHEN o jardineiro confirma o envio
THEN o WhatsApp e aberto com o numero do cliente pre-preenchido
AND a mensagem aparece pronta para enviar

AC-3: Atualizar status para "Enviado"
GIVEN o WhatsApp foi aberto com sucesso
WHEN o deep link e ativado
THEN o status do orcamento muda automaticamente para "Enviado"
AND a data de envio e registrada

AC-4: Cliente sem telefone
GIVEN o cliente nao tem telefone cadastrado
WHEN o jardineiro toca em "Enviar por WhatsApp"
THEN uma mensagem de alerta e exibida informando que o cliente nao tem telefone
```

[Source: PRD Section EPIC-04, Story 4.2]
