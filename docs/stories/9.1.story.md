## Story

**As a** jardineiro com plano Equipe,
**I want** convidar membros para minha equipe via telefone,
**so that** eu possa delegar servicos e gerenciar minha equipe no app.

## Acceptance Criteria

```gherkin
AC-1: Convidar novo membro (Happy Path)
GIVEN o admin esta na tela "Minha Equipe"
WHEN ele toca em "Convidar membro" e preenche nome e telefone
THEN o sistema cria um TeamMember com status=invited
AND envia um convite via SMS (Linking.openURL) com link de cadastro
AND o membro aparece na lista com badge "Convidado"

AC-2: Membro aceita convite
GIVEN um membro recebeu o convite via SMS
WHEN ele cria sua conta no app e e vinculado a equipe
THEN seu status e atualizado para active
AND ele ve apenas os servicos atribuidos a ele
AND ele NAO ve dados financeiros

AC-3: Gate de plano
GIVEN um usuario com plano basico (nao Equipe)
WHEN ele acessa a secao "Minha Equipe"
THEN o sistema exibe: "Faca upgrade para o plano Equipe para adicionar membros"
AND o botao "Convidar membro" esta desabilitado

AC-4: Remover membro
GIVEN o admin visualiza a lista de membros
WHEN ele toca em um membro e seleciona "Remover"
THEN o status do membro e alterado para inactive
AND os servicos futuros atribuidos a ele sao desatribuidos
AND o membro some da lista ativa

AC-5: Criar equipe automaticamente
GIVEN um usuario que acabou de fazer upgrade para o plano Equipe
WHEN o app detecta o novo plano
THEN uma Team e criada automaticamente com o usuario como admin
```
