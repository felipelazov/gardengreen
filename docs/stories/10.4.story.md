## Story

**As a** jardineiro usando o dashboard web,
**I want** ver minha agenda em formato de calendario mensal/semanal/diario,
**so that** eu tenha uma visao completa dos meus servicos agendados e possa reorganizar facilmente.

## Acceptance Criteria

```gherkin
AC-1: Calendario Mensal
GIVEN o usuario acessa "/agenda"
WHEN a pagina carrega
THEN ele ve um calendario mensal com os servicos marcados como chips coloridos
AND cada chip mostra: horario + nome do cliente
AND cores indicam status: verde=agendado, azul=em andamento, cinza=concluido, vermelho=cancelado

AC-2: Visoes Dia/Semana/Mes
GIVEN o calendario exibido
WHEN o usuario alterna entre visoes (Dia, Semana, Mes)
THEN a visualizacao muda correspondentemente
AND a visao semanal mostra timeline vertical com slots de horario
AND a visao diaria mostra detalhes expandidos de cada servico

AC-3: Drag-and-Drop para Reagendar
GIVEN a visao semanal ou mensal
WHEN o usuario arrasta um servico para outro dia/horario
THEN um dialog de confirmacao aparece: "Reagendar {servico} de {dataOriginal} para {novaData}?"
AND ao confirmar, o servico e atualizado no Supabase
AND o calendario atualiza em tempo real

AC-4: Criar Servico pelo Calendario
GIVEN qualquer visao do calendario
WHEN o usuario clica em um slot vazio ou no botao "+"
THEN abre um dialog para agendar novo servico
AND o formulario pre-preenche a data/hora clicada
AND permite selecionar cliente (combobox) e tipo de servico

AC-5: Detalhes do Servico
GIVEN um servico no calendario
WHEN o usuario clica no chip
THEN abre um popover com detalhes: cliente, endereco, horario, preco, notas
AND botoes: Marcar Concluido, Editar, Cancelar, Abrir no WhatsApp
```
